# Python - If/Else, Loops, Functions

This directory contains Python scripts that demonstrate basic concepts like conditional statements, loops, and functions.

## Scripts

### Conditional Statements

* `0-positive_or_negative.py`: Generates a random number and prints whether it is positive, negative, or zero.
* `1-last_digit.py`: Generates a random number and prints the last digit with specific conditions.

### Loops

* `2-print_alphabet.py`: Prints the alphabet in lowercase without using variables.
* `3-print_alphabt.py`: Prints the alphabet in lowercase except 'q' and 'e'.
* `4-print_hexa.py`: Prints numbers from 0 to 98 in decimal and hexadecimal.
* `5-print_comb2.py`: Prints numbers from 00 to 99 separated by commas.
* `6-print_comb3.py`: Prints all possible different combinations of two digits.

### Functions

* `7-islower.py`: Function that checks if a character is lowercase.
* `8-uppercase.py`: Function that prints a string in uppercase.
* `9-print_last_digit.py`: Function that prints the last digit of a number.
* `10-add.py`: Function that adds two integers and returns the result.
* `11-pow.py`: Function that computes a to the power of b and returns the result.
* `12-fizzbuzz.py`: Function that prints the numbers from 1 to 100 with FizzBuzz logic.

## Learning Objectives

* Understanding conditional statements (if/else)
* Working with loops (for, while)
* Creating and using functions
* String manipulation and formatting
* ASCII character operations
* Mathematical operations in Python
