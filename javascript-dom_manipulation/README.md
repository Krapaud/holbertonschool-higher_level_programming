# JavaScript - DOM Manipulation

This project contains JavaScript scripts that manipulate the DOM (Document Object Model).

## Files

- `0-script.js` - Updates the text color of the header element to red
- `1-script.js` - Updates the text color of the header element to red when clicking on an element
- `2-script.js` - Adds a CSS class to the header element when clicking on an element
- `3-script.js` - Toggles classes on the header element
- `4-script.js` - Adds list items dynamically
- `5-script.js` - Updates the text of the header element
- `6-script.js` - Fetches and displays a Star Wars character name
- `7-script.js` - Fetches and lists Star Wars movie titles
- `8-script.js` - Fetches and displays a translation of "hello"
